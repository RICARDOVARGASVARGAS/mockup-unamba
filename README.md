# mockup-unamba

Workspace con los mockups de UNAMBA. Abre `index.html` para elegir un proyecto.

## Proyectos

- **`pagina-web/`** — Mockup de la nueva web institucional de la Facultad de Administración.
- **`docentrack/`** — DocenTrack: registro de horas lectivas docentes con huella digital (kiosko + panel admin).

Cada proyecto es independiente y autocontenido, con su propio `CLAUDE.md`.
